Advanced dplyr & ggplot2

A structured, six-module curriculum for moving beyond introductory R into professional-grade data wrangling and visualization.

Module 1 — Tidyverse Foundations dplyr ggplot2

Prerequisite · ~2 hours
  • Native pipe |> and magrittr %>% — when to use each
  • Tidy data principles: pivot_longer(), pivot_wider(), separate(), unite()
  • Factor handling with forcats: fct_infreq(), fct_relevel(), fct_lump_n()
  • Date/time workflows with lubridate: parsing, arithmetic, floor_date, quarter
📄 01-tidyverse-foundations.R

Module 2 — Advanced dplyr dplyr

Core module · ~4 hours
  • Grouped operations: group_by(), summarise(), multi-level grouping
  • Window functions: lag(), lead(), cumsum(), percent_rank(), ntile()
  • All join types — inner, left, anti, semi — and non-equi joins with join_by()
  • Rolling / nearest joins with closest()
  • Rowwise workflows: rowwise() + c_across()
  • List-columns: nest() / unnest(), map(), per-group modelling
  • Tidy evaluation: {{ }}, across(), .data[[]], pick(), enquos()
  • Performance: dtplyr / data.table back end, profiling tips
📄 02-advanced-dplyr.R

Module 3 — Advanced ggplot2 ggplot2

Core module · ~4 hours
  • Layered grammar deep-dive: geoms, stats, scales, coordinates, facets
  • Scale mastery: log axes, dollar_format(), percent_format(), continuous size/alpha
  • facet_wrap() with free scales; facet_grid() for two-variable panels
  • Annotation strategy: annotate(), arrows, rectangles, callout labels
  • Non-overlapping labels with ggrepel
  • Spotlighting subsets with gghighlight
  • Rich distribution plots with ggdist: stat_halfeye(), stat_dots()
  • Multi-panel layouts with patchwork
  • Building a custom publication theme; theme_set()
  • Custom Stat via ggproto()
📄 03-advanced-ggplot2.R

Module 4 — Project: Health Economics Analysis Project

Applied project · ~3 hours
  • dplyr focus: year-over-year window functions, rolling averages, reusable tidy-eval summaries, across() for multi-column stats, nested OLS models per country
  • ggplot2 focus: four-panel patchwork dashboard, ribbon confidence bands, labelled scatter with ggrepel, custom theme_health()
  • Covers: 30-country × 15-year panel data, health expenditure, life expectancy, infant mortality, pricing
📄 04-project-health-economics.R

Module 5 — Project: Insurance Claims Analysis Project

Applied project · ~3 hours
  • dplyr focus: non-equi joins for severity banding, rowwise() risk scoring, tidy-eval summarise_by() helper, window functions for frequency trends, list-column severity models
  • ggplot2 focus: ggdist raincloud plots, loss-ratio heatmap, gghighlight spotlight, stacked bar chart with labels, pricing adequacy scatter
  • Covers: 5,000-policy × 1,800-claim auto insurance portfolio simulation
📄 05-project-insurance.R

Module 6 — Capstone: Before/After Portfolio dplyr ggplot2

Capstone · ~2 hours
  • Before/after data cleaning: multi-format date parsing, type coercion, string normalisation, separate()
  • Before/after chart redesigns: pie → labelled bar; rainbow scatter → outcome-coloured with regression; plain boxplot → raincloud
  • Design principles documented in code: colour strategy, chart-type selection, annotation over legends, performance tradeoffs
  • Reproducibility checklist: seeds, renv, immutable raw data, Quarto rendering
📄 06-capstone.R
